Простой скрипт на питоне, который позволяет вам записывать тайминги текстом ваших любимых песен!
===========
Теперь вам не нужно беспокоиться о том, как синхронизировать музыку с текстом песни.

Как установить
===========
* Загрузите Python 3.11 (более новые или старые версии могут поддерживаться, но это не было проверено).
* Загрузите последнюю версию или клонируйте репозиторий
* Возможно вам так-же понадобавиться установить ffmpeg в path (если вы в Используете Windows)
* Опционально: Установить Visual Studio Code, для более эстетичного отображения текста
* <s> Установите зависимости при помощи команды: `pip install -r requirements.txt`</s><br>Теперь при запуске скрипта и отсутсвии зависимостей скрипт сам 

Как использовать:
------
1. Запустите main.py любым удобным вам спосообом.
2. После первого запуска появиться инструкция, что делать дальше)
3. Наслаждайтесь!
